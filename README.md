# Kanagawa Like for Zed

A Kanagawa-inspired theme for Zed, tuned to feel closer to popular VS Code and Neovim Kanagawa variants.

## Description

Kanagawa Like is a custom dark theme for Zed built to capture the feel of Kanagawa setups commonly seen in VS Code and Neovim, with a few intentional color adjustments to better fit Zed.

## Included themes

- Kanagawa Like Wave
- Kanagawa Like Dragon

## Installation

1. Open Zed.
2. Run `zed: install dev extension`.
3. Select this repository folder.
4. Open `theme selector: toggle`.
5. Choose `Kanagawa Like Wave` or `Kanagawa Like Dragon`.

## Settings example

```json
{
  "theme": {
    "mode": "system",
    "light": "One Light",
    "dark": "Kanagawa Like Wave"
  }
}
```

## Notes

- This is not intended to be a strict 1:1 port of Kanagawa.nvim.
- Zed does not always map syntax tokens exactly like VS Code or Neovim.
- Some colors were adjusted to better match real-world Kanagawa screenshots.
